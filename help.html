First Time Installation

Here's the path to success for the first time installation:

  1. Watch the setup video {setup_video}. That may be enough to complete the installation! 
  2. Set up and connect everything on a table or desktop and provision the Brain:
    • Get your phone. You'll need it to connect the Brain to WiFi.
    • Place the Brain and all of the sensors on a table. Make sure there is an outlet nearby.
    • Connect the temperature probe and all of the sensors to their respective connectors on the harness.
    • Plug in the Brain and follow the instructions below for Provisioning.
  3. Follow the instructions below for Setting Up the Web App for the First Time.
  4. Follow the instructions below for Installing in your Kegerator or Keezer. You may want to "try-out" the installation location, inside or outside the kegerator or keezer, before committing to the full installation. You want to make sure the web app stays connected when the Brain is in its final installation location.

Provisioning (Connecting to WiFi)

The very first time the KegLevel Brain is powered on, it creates its own WiFi network called KegLevel-Brain. When you connect to that network with your computer or phone, a setup page is automatically loaded (Captive Portal). This allows you to select the mode you will use: Network or Standalone. The Network  mode provides the most robust features and requires the Brain to be connected to your WiFi network. The Standalone mode provides all but a few features and operates fully isolated from your WiFi network. After provisioning, you can switch between modes at any time from Settings > System. But to connect to the Brain the very first time, you must decide which mode you'll use:

Network Mode vs. Standalone Mode

Feature Network Mode Standalone Mode
Web interface Available to any number of browsers on your WiFi network simultaneously Available to one browser at-a-time at 192.168.4.1
Email notifications Available Not available
Firmware updates OTA (Over-the-Air) Available Not available
Internet required Yes (your WiFi network must have Internet) No

First Time Provisioning

  1. Power on the KegLevel Brain.
  2. Navigate to the WiFi settings on your phone or computer. Momentarily you should see KegLevel-Brain in the list of WiFi networks. Select it. 
  3. A "captive portal" browser window should open; it may take 5-15 seconds so be patient.

    IMPORTANT! If you have trouble connecting to the "captive portal", your phone or computer might be trying to be helpful. Sometimes they will bypass a network selection that does not have Internet access (the KegLevel-Brain network) and will try to auto-connect to a network that does (your WiFi network). If this happens, try this:

    • Turn off auto-connect / auto-join or select "forget" for your network until provisioning is complete.
    • Select Airplane Mode until provisioning is complete. 
    • If you are still unable to connect to the captive portal, contact keglevelmonitor@gmail.com for help.

  4. Select the desired mode: Network or Standalone.

If you selected Network:

  1. Select the desired network in the drop down list.
  2. Enter the network password and click Connect.
  3. You will see a Connecting to {network}... message. Be patient. This may take 30-60 seconds.
  4. When the connection is complete, a Connected! message appears with an IP address. That is the IP address your network has assigned to the Brain. Take a screen snapshot or write down the IP address.
  5. Reconnect your computer or phone to your network.
  6. Launch a browser window, type the Brain's IP address into the browser's address bar, and press Enter. The main TAPS screen should be shown with 5 taps "Offline". Follow the instructions to set up the web app for the first time.
    If your WiFi credentials change (network ID or password) you'll need to use the WiFi Reset button to provision the KegLevel Brain again. Refer to WiFi Reset.

    If your WiFi network router reassigns an IP address to the Brain (rare) you'll need to do one of these:
        - find the newly-assigned IP address using your router's configuration pages
        - reset the WiFi and provision again; refer to WiFi Reset
    If desired, you can use your network router's configuration pages to assign a static IP address to the Brain. This will ensure the IP address is not reassigned on the network.

If you selected Standalone:

  1. You will see the message Standalone Mode Enabled. The Brain has created its own WiFi network called KegLevel-Brain and is broadcasting its website at 192.168.4.1.
  2. Close the captive portal window on your device. Return to your device's WiFi Settings screen.
    If you receive a popup asking, Use Without Internet? answer Yes. The Brain created its own WiFi network and does not have access to your Internet-connected WiFi network. Therefore it is not connected to the Internet..
  3. Select the KegLevel-Brain network from the list of WiFi networks on your device, if not already selected.
  4. Launch a browser window, type 192.168.4.1 into the browser's address bar, and press Enter. The main TAPS screen should be shown with 5 taps "Offline". Follow the instructions to set up the web app for the first time.

Setting Up the Web App for the First Time

After provisioning, follow these steps to set up the web app:

  1. Select the basic settings  —
    • Click SETTINGS.
    • Set the number of taps you have connected. Only active taps will appear on the Taps screen.
    • Select the desired measurement units, Metric (liters/Celcius) or Imperial (gallons/Fahrenheit).
    • Enable or disable Leak Detection. When On, if the brain senses a very slow flow rate that is less than a pour, it will highlight the tap with "Leak?" and if notifications are configured, will email a notification.
  2. Add your beverages — Click BEVERAGES and add each beer, cider, or other beverage you have on tap. Enter the name, ABV, IBU, and SRM (color) for each. You can edit or delete the default beverages.
  3. Add your kegs — Click KEGS and add a keg for each physical keg used in your kegerator. Set the empty weight (tare) and maximum capacity for each keg. If the keg has a beverage in it, select a beverage. You can edit or delete the default kegs.
    It is helpful to label your kegs with the keg name and tare weight. A paint pen works great.

    The first time you install the KegLevel Brain you will probably have kegs already in the kegerator. For now just estimate the tare weight and current weight/volume of the kegs. Until those kegs are replaced with kegs for which you have a measured tare weight and a measured filled weight, the volume shown for the kegs may not be accurate.
  4. Assign kegs to taps — Click TAPS. To assign a keg, click on a tap and select the keg from the list. 
  5. Calibrate the sensors (optional) — The sensors are pre-calibrated and the calibration K-factors are entered into the Brain before the system is shipped to you. Just be sure to match the sensor to the tap number and sensor calibration should not be necessary.

Installing in your Kegerator or Keezer

Sensor Placement

Sensors purchased directly from KegLevel Monitor are pre-tested and pre-calibrated. Install one flow sensor inline with each beverage line you want to monitor. Place each sensor as close to the OUT post (liquid post) of the keg as possible. This reduces agitation and ensures the most accurate measurement of dispensed volume. Depending on your kegerator, you may need a short piece of beverage tubing and a clamp to splice the sensor into the beverage line. Note that each sensor's calibration "K-factor" has been entered into the KegLevel Brain and is written on the sensor for reference.

KegLevel Brain Placement

When determining the placement of the KegLevel Brain you must ensure:

  • the wiring harness will reach the farthest keg without stressing the cabling;
  • the Brain is placed such that spillage will not immerse it;
  • there is good WiFi signal from your network at the kegerator if you plan to use Network mode;
  • there is an available outlet nearby to plug in the Brain;
  • there is room to rotate kegs into/out of your kegerator without interfering with the cabling.

The Brain can be placed inside or outside the kegerator:

  • If placed inside, the USB cable that powers it must be run to the outside. Be aware the WiFi signal at the Brain may be weakened when installed inside, which may cause the system to feel sluggish. This issue is location-specific and should be tested before the device is fully installed inside. 
  • If placed outside, the wiring harness(es) that connect to the sensors must be run to the inside. This is the best placement option if the WiFi network signal near the kegerator is weak.

Ideas for routing the cabling:

  • Home-built Keezer — If the keezer has a wood collar, drill a hole through the collar and pass the cabling through. Seal with plumber's putty. If there is no collar, two other options are: (1) Pass the cabling between the door and the freezer body, probably near the back by the hinges. The door gasket should be flexible enough to allow a good seal with the USB cable or wiring harnesses in place. (2) Drill a hole through the lid close to the hinge side and pass the cabling through. Seal with plumber's putty. 
  • Home-built Kegerator — Drill a hole through the door close to the hinge side and pass the cabling through. Seal with plumber's putty. Or pass the cabling between the door and the refrigerator body, probably on the hinge side. 
  • Commercial Kegerator — Drilling a hole through the body of a commercial kegerator is NOT recommended. Install the Brain inside the kegerator, and pass the USB cable that powers it between the door and the refrigerator body. The hinge side might be best. The door gasket should be flexible enough to allow a good seal with the USB cable in place.

Wiring harness

The wiring harness is pre-assembled, pre-tested and labeled:

  • Match the temperature probe connector with the mating harness connector and plug it in firmly. Place the probe inside the kegerator where it will not contact the kegs or the sides of the kegerator.  Placing the probe inside a small housing with numerous holes in it allows the sensor to free-float without touching anything inside the kegerator. A pickleball is ideal for this application.
  • Match each flow sensor connector to the corresponding tap number on the harness. Plug in each connector firmly.
  • IMPORTANT! If you purchase sensors directly from KegLevel Monitor, the sensors are pre-tested and pre-calibrated. If you purchase them from Amazon or another online merchant, beware the wiring of the sensor connector MAY OR MAY NOT MATCH the wiring of the harness connector. There appears to be no industry standard for wire colors of the sensor connector. What matters is the function of the wire colors at the Brain's harness and how that relates to the function of the wires at the sensor connector:
    • Brain harness YELLOW = VCC (supply voltage +5V to the sensor)
    • Brain harness BLACK = GND (ground to the sensor)
    • Brain harness RED = SIGNAL (signal from the sensor to the Brain)

If the wiring of the sensor connector is not correct, it is easy to fix. At the sensor connector, using a small flat-blade jeweler's screwdriver or a safety pin, push in the pin's retention clip while gently pulling out the pin. Do the same for the other wires that do not match the Brain harness. Reinstall the pins in the correct location being sure the pin is reinstalled with the retention pin in the same orientation as when it was removed, and be sure it "clicks" into place. Gently pull on the pin to make sure it does not slide out on its own. If it does, pull out the pin and using the screwdriver or safety pin, gently bend out the retention pin so it protrudes slightly. It only needs to protrude a very small amount, less than 1 mm.

Validating the Installation

Use a keg filled with sanitizing solution to test the installation, as this also sanitizes the flow sensors and flushes out the beer lines.

TAPS Screen

The TAPS screen is the main dashboard. It shows a card for each active tap with real-time information. Taps are assigned to a keg, and a keg contains all of the beverage information.

Tap Cards

Each tap card displays:

  • Tap number — Tap 1 through Tap 5
  • Beverage name — The name of the beverage in the keg
  • ABV, IBU, SRM — Beverage stats
  • Keg level — Visual indicator of the volume of beverage remaining in the keg
  • Keg name — The name of the keg assigned to the tap
  • Volume remaining — The volume of beverage remaining in the keg
  • Pour indicator — Shows Pouring during a pour; Idle if no activity; Offline if no keg assigned; Leak? if a possible leak has been detected

Tap Card Actions

Click the keg icon on a tap card to open the keg selection menu. From there you can:

  • Assign a keg to the tap
  • Mark the current keg as Kicked:
    • Keg Kicked - Calibrate will perform a full-volume calibration of the keg. Only do this if (1) the keg's tare weight and filled weight were measured and entered before the keg was assigned to the tap and (2) the keg was fully dispensed from this tap. Follow the prompt to complete the action.
    • Keg Kicked - Mark Empty will zero the keg's volume, assign Empty to the keg, and remove the keg assignment from the tap. Follow the prompt to complete the action.
  • Mark the tap Offline. Removes the keg assignment from the tap but does not make any changes to the keg settings.

Temperature

The current kegerator temperature is shown in the header bar at the top left. It updates automatically. Blank if the temperature probe is disconnected or faulty.

KEGS Screen

The KEGS screen is your keg inventory. Each row shows a keg with its stats and action buttons. Each keg used with your kegerator should be added here.

Adding a Keg

Click + ADD NEW to create a new keg:

  • Keg Name — Enter the keg name, maximum 24 characters, or keep the default name
  • Beverage — Select which beverage is in this keg
  • Empty Weight (Tare) — Enter the empty (tare) weight of the keg (only needs to be done once)
  • Max Capacity — Enter the maximum capacity (volume) the keg holds (only needs to be done once)
  • Filled Weight (Total) — Enter the filled (total) of the keg at the time it is placed in the kegerator; this will be used to calculate the Volume at Fill
  • Volume at Fill — Automatically calculated from the empty and filled weights, and assumes a standard specific gravity of 1.014
  • Tap assignment — Assign this keg to a tap, or leave it Unassigned if a tap is not available

Editing a Keg

Click the edit button or click on a keg row to edit any of the settings. Saved changes are immediately reflected on the tap card if the keg is assigned to a tap. 

Deleting a Keg

Click the delete button on the keg row. You will be asked to confirm. If a keg is assigned to a tap, deleting the keg removes the assignment and sets the tap to Offline. Deleting a keg does not affect Beverages.

BEVERAGES Screen

The BEVERAGES screen is your beverage library. Each row shows a beverage metrics, and action buttons. Every unique beer, cider, mead, or other drink you serve should be added here. 

Adding a Beverage

Click + ADD NEW to create a new beverage:

  • Name — The beverage name (e.g. "American IPA")
  • ABV — Alcohol by volume (e.g. 6.5)
  • IBU — International Bitterness Units (e.g. 55)
  • SRM — Standard Reference Method color value (e.g. 8). A color swatch preview is shown as you adjust this value.

Editing a Beverage

Click the edit button or click on a beverage row to edit any of the settings. Saved changes are immediately reflected everywhere the beverage is used (tap cards, keg assignments, BatchFlow cards).

Deleting a Beverage

Click the delete button on the beverage row. If the beverage is currently assigned to a keg, you will be notified and must remove it from the keg first.

SETTINGS: SYSTEM

Active Taps

Set the number of taps you have sensors connected to. The Brain supports up to 5 taps. Only active taps are shown on the Taps screen and processed for flow detection and optional leak detection.

Units

Choose Metric (liters, Celsius) or Imperial (gallons, Fahrenheit, US Imperial). This affects all volume and temperature displays throughout the app except the volumes on the CALIBRATION screen (to support the industry-standard treatment of flow sensor data in metric volumes).

Leak Detection

Enable or disable Leak Detection. When enabled, if the system detects a very slow flow rate over a 15-20 second time period (far slower than a pour), the system generates a "Leak?" condition: the tap is highlighted and shows "Leak?". If the NOTIFICATIONS screen has valid email settings, a Conditional Notification is emailed.

Network Mode / Standalone Mode

Switch between Network Mode and Standalone Mode. See Provisioning for details on the differences between modes. Switching modes forces the Brain to reboot immediately and enter the Provisioning mode.

The button for the current mode is always disabled.

SETTINGS: NOTIFICATIONS

The KegLevel Brain can send email notifications using free email service at mailgun.com. A free Mailgun account is sufficient for typical usage.

Notifications require Network Mode with Internet access. Notifications are not available in Standalone Mode.

Mailgun Credentials

First set up an account (free) at Mailgun.com, then configure the Brain for notifications:

  1. Navigate to mailgun.com and click Start for free to create a new account. If there is an "Add payment info now" checkbox, be sure to uncheck it. Enter your desired password and confirm it, then click Create New Account.
  2. At the dashboard, you'll see a notice that you must verify your email address first. Open your email, find the verification email from Mailgun (it may be in your Spam folder), and click Verify your Email Address.
  3. Verifying email address calls up a window for you to also verify through SMS (text) as well. This is to prevent abuse of their system. Follow the instructions to validate via text.
  4. When the text has been validated, the Mailgun login window will appear. Log in and you'll see the Create an API key helper. Enter a name for the API key (such as KegLevel Brain) and click Create API Key.
  5. When the API key has been created, the helper called Copy and run the code in your IDE expands. You only care about the two Mailgun credentials: API Key and the Sandbox domain. Copy them and paste them into a text editor and save for safekeeping. 
  6. Copy and paste each of the credentials into the corresponding fields on the NOTIFICATIONS screen.
  7. Enter your email address and click SAVE.
    You MUST use the same email address as your Mailgun account. Otherwise Mailgun will ignore the requests.
  8. Click TEST SEND to verify the configuration. You should receive a test email.

Scheduled Notifications

Select the Notification Frequency to receive periodic email reports summarizing your keg levels and temperature.

Conditional Notifications

Set thresholds for a Conditional report to be sent when the threshold is crossed:

  • Low Volume — Triggered when any keg drops below the set volume. Slide to OFF to disable.
  • Low Temp — Triggered when the kegerator temperature drops below the set threshold. Useful for detecting cooling control malfunctions.
  • High Temp — Triggered when the kegerator temperature rises above the set threshold. Useful for detecting door-left-open or cooling failure.

SETTINGS: UPDATES

The KegLevel Brain supports over-the-air (OTA) firmware updates. When a new version is available, you can install it directly from this screen without disconnecting anything.

OTA updates require Network Mode with Internet access. They are not available in Standalone Mode.

Checking for Updates

Click CHECK to query the update server. The current firmware version is shown at the top of the screen. If a newer version is available, the updates are listed and the INSTALL button will become active.

Installing an Update

Click INSTALL to begin the update. A progress bar will show the download and installation progress. The update log shows detailed status messages. After installation completes, click RELOAD to refresh the browser window for the new firmware. 

Do not power off the Brain during a firmware update. The web UI is included with the firmware and updates automatically.

SETTINGS: CALIBRATION

Sensors purchased from KegLevel Monitor are calibrated using pour calibration. The K-factor is written on the sensor. This K-factor should be used until an entire keg has been dispensed through the sensor, then the Keg Kicked calibration should be performed. This will provide the most accurate measurements.

  • Pour Calibration — A small volume is poured, measured, and entered. The system calculates the K-factor based on this pour. This is done at the CALIBRATION screen. This method is fast but not highly accurate. 
  • Keg Kicked Calibration — An entire keg is dispensed from full to empty. The system calculates the K-factor based on the total number of pulses counted and the starting volume of the kicked keg. This is done at the TAPS screen. This method is not fast but highly accurate.

Workflow for Most Accurate Measurements

  1. At the CALIBRATION screen, for the tap/sensor to be calibrated, enter the K-factor written on the sensor. If the K-factor is not written on the sensor, perform a Pour Calibration and enter that K-factor.
  2. Weigh an empty keg (with the lid on) and write that empty weight on the keg.
  3. Fill the keg and weigh it, and write that filled weight on the keg.
  4. Install the keg in the kegerator and connect the tap.
  5. At the KEGS screen, select the keg, enter the tare weight (only needs to be done once), the max capacity of the keg (only needs to be done once), and the filled weight. The Brain will calculate the volume. Assign the keg to the tap and Save.
  6. When the entire keg has been dispensed (keg kicks), at the TAPS screen, click on the tap card and select [ Keg Kicked - Calibrate ]. Review the metrics for the keg. You'll see "CALCULATED NEW K-FACTOR" which is based on the total number of pulses counted for the dispensed volume of the keg. If the metrics look correct, click the checkbox "This keg is empty and was fully dispensed from this tap." to confirm. Click SAVE CALIBRATION & RESET to save the new K-factor for the tap/sensor.
  7. This process usually needs to be done only once. As long as the keg kicks within a pint or two of the zero measurement the calibration is accurate. If necessary the Keg Kicked calibration can be repeated.
The Temp sensor is factory-calibrated and requires no user calibration.

How to perform the Pour Calibration

  1. Go to Settings > CALIBRATION.
  2. Select the tap you want to calibrate. The tap button will highlight and the Brain gets ready to count pulses.
  3. If the sensor has already been calibrated using this method, the K-factor should be written on the sensor. You can simply enter this as the New K-factor and click SAVE CALIBRATION, or you can continue with the calibration below.
  4. Pour a known volume of liquid through that tap into a measuring container. Pour only from the selected tap.
  5. Use the Measured Volume slider (or +/− buttons) to enter the exact volume you poured.
  6. The New K-Factor is calculated automatically from the measured volume and pulse count.
  7. Click SAVE CALIBRATION to apply the new K-factor.

Other Calibration Actions

  • RESET PULSES — Clears the pulse counter to zero so you can start a fresh calibration pour.
  • SET TO DEFAULT — Resets the K-factor to the factory default (2900 pulses per liter).

Tips for Best Results

  • Use a graduated measuring container for accurate volume measurement.
  • Pour at least 500 mL for a reliable calibration — larger volumes produce more accurate results.
  • Pour at a normal pace, similar to how you would pour a glass of beer.
  • If the Deduct volume from Inventory checkbox is checked, the poured volume will be subtracted from the keg's current level, using the current K-factor to calculate the volume poured from the keg.

BatchFlow has its own help system. Click the ? button inside the BatchFlow web app for detailed instructions on using the workflow board, adding batches, moving cards between columns, and editing beverages.

Troubleshooting

LED Indicator

LED Behavior Meaning
Blinking slowly Brain is provisioning; the WiFi mode has not been selected. Refer to Provisioning. 
Solid on Brain is connected to WiFi (either Network or Standalone) and is operating normally
LED off Brain has lost power or hard crashed — unplug and plug in again

Common Issues

Problem Possible Cause & Fix
Web page loads from phone but not from PC Windows Firewall or a proxy may be intercepting traffic on port 80. Try disabling the firewall temporarily to test.
No pulses counted during pour Check the sensor wiring. Ensure the sensor connector is fully seated in the wiring harness. Try a different tap to isolate whether the issue is the sensor or the wiring.
Temperature shows "—" or no reading The temperature probe may be disconnected. Check the sensor connector on the wiring harness.
Can't connect to KegLevel-Brain WiFi The Brain may not be in Standalone mode, or it may still be booting. Wait 30 seconds after powering on and try again. If the issue persists, provision again. Refer to WiFi Reset.
Notifications not sending Verify that the Brain is in Network mode with Internet access. Double-check the Mailgun API key, domain, and email addresses on the Notifications settings screen. Use TEST SEND to diagnose. Check the Spam folder in your email program.
OTA update fails Ensure the Brain has a stable Internet connection. Do not power off during the update. If the update fails, the previous firmware should still be running — try again.

WiFi Reset

If you cannot locate the KegLevel Brain either on your WiFi network (Network mode) or at 192.168.4.1 (Standalone mode) you have a few options. If you set up in Network mode, you can use your network router's configuration pages to find all of the devices connected to WiFi on the network. You can also use a app such as Fing on your phone to locate devices on your WiFi network.

If those don't work, you can "force" provisioning using the WiFi Reset button. Unplug the Brain, remove the cover and locate the black reset button mounted on the larger PC board (not the white button on the Pico). Plug in the Brain and give it 10 seconds to boot up. Press and hold the WiFi reset button for about 5 seconds, until the LED switches from rapid flashing to steady on, then release the button. The LED should begin blinking slowly indicating it is in provisioning mode. Refer to Provisioning.
